# Tic-Tac-Toe

A browser-based tic-tac-toe game built with HTML, CSS, and vanilla JavaScript.

## How to play

Open `index.html` in any browser — no build step needed.

- Two players take turns clicking empty cells
- Player X uses the boy image, Player O uses the kid image
- The game announces the winner or a draw automatically
- Click **Restart** to reset the board

## Project structure

```
tic-tac-toe/
├── index.html       # Game board and markup
├── css/
│   └── style.css    # Layout and cell styling
├── js/
│   └── game.js      # Game logic, win detection, restart
└── img/             # Player images
```

## What I practiced

- DOM manipulation with vanilla JS
- Event listeners and conditional logic
- CSS table layout and centering
